# DecisionFlow - 智能决策助手

基于AHP + TOPSIS混合模型的交互式多准则决策辅助Web应用。

## 功能特性

### 🎯 核心功能
- **问题定义**：自定义决策问题、评价准则和备选方案
- **AHP权重分配**：直观的成对比较界面，实时一致性检验
- **方案评分**：矩阵式评分表，支持1-5分评分
- **TOPSIS分析**：自动计算方案排名和相对贴近度
- **可视化展示**：丰富的图表展示权重分布和排名结果

### 📊 可视化图表
- 权重分布柱状图
- 方案排名横向条形图
- 前三名方案雷达图
- 评分对比分析图

### 🚀 技术特色
- 使用最新的React 19 + TypeScript + Vite技术栈
- Material-UI现代化设计
- 响应式设计，支持桌面和移动设备
- 实时计算和动画效果

## 快速开始

### 环境要求
- Node.js 18+
- pnpm (推荐) 或 npm

### 安装和运行

```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm dev

# 构建生产版本
pnpm build

# 预览生产版本
pnpm preview
```

### 使用步骤

1. **问题定义**
   - 输入决策问题标题和描述
   - 添加评价准则（最多8个）
   - 添加备选方案（最多20个）

2. **AHP权重分配**
   - 通过滑动条进行成对比较
   - 系统实时计算权重和一致性比率
   - 确保一致性比率低于0.1以获得可靠结果

3. **方案评分**
   - 为每个方案在各个准则上评分（1-5分）
   - 1分：很差，5分：优秀

4. **结果分析**
   - 查看方案综合排名
   - 分析权重分布和评分情况
   - 导出结果报告

## 算法原理

### AHP (层次分析法)
1. 构建成对比较矩阵
2. 计算特征向量得到权重
3. 进行一致性检验
4. 确保CR < 0.1

### TOPSIS (优劣解距离法)
1. 数据标准化处理
2. 构建加权标准化矩阵
3. 确定理想解和负理想解
4. 计算欧氏距离
5. 计算相对贴近度
6. 排序方案
